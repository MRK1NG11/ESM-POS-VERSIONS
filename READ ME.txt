INTRODUCTION
------------
Welcome to the Eat Sum Meat Point of Sale (POS) system. This software is a 
custom-built, high-speed, touchscreen-optimized cash register application. 
It is designed to process sales quickly, manage live inventory, track staff 
shifts, and provide secure managerial oversight without relying on a constant 
internet connection for core functions.

1. HOW TO RUN THE SYSTEM
=============================================================================
For Windows Terminals (Executable):
1. Locate the "EatSumMeatPOS.exe" file on the desktop or target folder.
2. Double-click to launch. The system will automatically check for updates 
   and boot into the login screen.

For Source Code (Python Developers):
1. Ensure Python 3.8+ is installed.
2. Install the required module by running: pip install requests
3. Run the script: python EatSumMeatPOS.py

DEFAULT LOGIN CREDENTIALS:
- Cashiers: CSH1, CSH2
- Manager/Admin: ADMIN
- Admin Override PIN: 1981 (Required for Voids, Markdowns, and Admin Panel)

2. INSTRUCTIONS FOR CORE FEATURES
=============================================================================
* PROCESSING A SALE:
  - Scan an item's barcode, or use the "🔍 MANUAL SELECT" button to browse 
    the visual catalog.
  - To increase quantity, simply scan/tap the item multiple times.

* MANUAL SELECT (MS):
  - Tap the "🔍 MANUAL SELECT" button below the barcode box.
  - A categorized grid will appear. You can filter by category (MEAT, SNACKS, 
    etc.) or use the search bars to type a name or barcode.

* PAYING OUT (CASH / CARD):
  - Use the quick-cash buttons (+R10, +R50, etc.) to tally the cash handed 
    by the customer.
  - Tap "PAY CASH" to finalize the transaction and calculate exact change.
  - If the customer is using a card machine, tap "CARD SALES".

* 2ND FUNCTIONS:
  - Tap "⚙️ 2ND FUNCTIONS" at the bottom left.
  - "⬇️ MARKDOWN PRICE": Allows a manager to temporarily change the price of 
    a selected item in the cart (requires Admin PIN).
  - "⌨️ OPEN NUMPAD": Opens a virtual number pad to manually type in 
    barcodes or amounts if the physical keyboard/scanner fails.

* ADMIN PANEL:
  - Accessible from the bottom right (Requires Admin PIN).
  - DASHBOARD: View today's gross revenue, cash vs. card splits, and a live 
    feed of today's receipts.
  - INVENTORY: View live stock. Select an item and click "+ ADD STOCK" to 
    receive new inventory.
  - STAFF LOGS: Track cashier login/logout times and total hours worked.
  - PRINT Z-REPORT: Runs the End-of-Day report, prints the total slip, and 
    packages the data for back-office synchronization.

4. THE AUTO-UPDATE SYSTEM
=============================================================================
This POS system features a built-in, zero-touch Auto-Updater linked to GitHub.

How it works:
1. Upon booting, the POS checks the cloud for a new version number.
2. If a newer version exists, a prompt appears asking to "UPDATE NOW" or "LATER".
3. Clicking "UPDATE NOW" downloads the latest EatSumMeatPOS.exe from the server.
4. The system utilizes a temporary batch script (.bat) to safely delete the old 
   executable, replace it with the new one, and instantly restart the POS.

There is no need for manual USB updates or IT intervention. As long as the till 
has an internet connection on boot, it will remain up to date.
