INTRODUCTION

Welcome to the Eat Sum Meat Point of Sale (POS) system. This software is a
custom-built, high-speed, touchscreen-optimized retail management suite.
Upgraded to the enterprise-grade DARK-PRO architecture, it is designed to
process high-volume sales quickly, prevent unauthorized access via Kiosk mode,
and provide secure, comprehensive managerial oversight without relying on a
constant internet connection.

HOW TO RUN THE SYSTEM
=============================================================================
    For Windows Terminals (Executable):

    Locate the "EatSumMeatPOS.exe" file on the desktop or target folder.

    Double-click to launch. The system will automatically check for updates
    and boot directly into Fullscreen Kiosk Mode.

For Source Code (Python Developers):

    Ensure Python 3.8+ is installed.

    Install the required module by running: pip install requests

    Run the script: python EatSumMeatPOS.py

DEFAULT LOGIN CREDENTIALS:

    Cashiers: CSH1, CSH2

    Manager/Admin: ADMIN

    Admin Override PIN: 1981 (Required for Markdowns, Pay Outs, and Admin Suite)

2. INSTRUCTIONS FOR CORE FEATURES
=============================================================================
    PROCESSING A SALE:

        Scan an item's barcode. The system will instantly clear the box for the
        next scan and display the item's name, stock, and price on the new
        "Verify Strip" below the scanner.

        To increase quantity, simply scan the item multiple times.

    MANUAL SELECT (MS):

        Tap the "MANUAL SELECT" button on the left grid.

        A categorized grid will appear. You can filter by category (MEAT, SNACKS,
        etc.) or use the search bars to type a name or barcode.

    PAYING OUT (CASH / CARD):

        Use the 8 quick-cash buttons (+R200 down to +R1) beneath the numpad to
        rapidly tally the exact cash handed over by the customer.

        The Total screen will glow RED if cash is DUE, and NEON GREEN for CHANGE.

        Tap "PAY CASH" to finalize the transaction and open the till.

        If the customer is using a card machine, tap "PAY CARD".

    MARKDOWNS & 2ND FUNCTIONS:

        Select an item in the cart and tap "MARK DOWN" to apply a custom price
        or percentage discount (Requires Admin PIN).

        Tap "2ND FUNC" to access Till Pay Outs. This allows managers to securely
        log cash removed from the till for store expenses (Requires Admin PIN).

    ADMIN SUITE (Requires Admin PIN):

        Accessible from the bottom left "⚙️ ADMIN VIP" button.

        DASHBOARD: View gross revenue splits and an expandable transaction ledger
        (click the arrow next to an order to see exactly what was bought).

        INVENTORY: View a live, scrollable data grid of all stock levels.

        STAFF LOGS: Track exact cashier login and logout times.

        PAY OUTS: View a secure audit trail of all cash removed from the till.

        HTML EXPORT: Generates a beautiful, professional daily executive report
        that opens instantly in your web browser.

        PRINT Z-REPORT: Runs the End-of-Day report, prints the total slip, and
        packages the data for back-office synchronization.

4. THE AUTO-UPDATE SYSTEM

This POS system features a built-in, zero-touch Auto-Updater linked to GitHub.

How it works:

    Upon booting, the POS checks the cloud for a new version number.

    If a newer version exists, a prompt appears asking to "UPDATE NOW" or "LATER".

    Clicking "UPDATE NOW" downloads the latest EatSumMeatPOS.exe from the server.

    The system utilizes a temporary batch script (.bat) to safely delete the old
    executable, replace it with the new one, and instantly restart the POS.

There is no need for manual USB updates or IT intervention. As long as the till
has an internet connection on boot, it will remain up to date.
